# Roadmap: From Founder to CEO

## Phase 1: The "Doer" (Months 0-3)
*   **Focus**: Sales & Shooting.
*   **Founder Role**: You are the Photographer, the Editor, and the Sales Rep.
*   **Goal**: $3k MRR.
*   **Tech**: Manual scripts (`process_leads.js`), local files.

## Phase 2: The "Editor" (Months 3-6)
*   **Focus**: Efficiency & Quality Control.
*   **Founder Role**: You hire a shooter (freelance) but you still edit. You are the "Quality Filter".
*   **Goal**: $8k MRR.
*   **Tech**: Cloud storage (Drive/Dropbox), automated invoicing (Stripe).

## Phase 3: The "Owner" (Months 6+)
*   **Focus**: Growth & Strategy.
*   **Founder Role**: You hire an editor (or use AI editing tools). You focus purely on partnerships and expansion.
*   **Goal**: $15k+ MRR.
*   **Tech**: Full CRM (HubSpot), AI Agents for outreach.

## Future Tech Roadmap (Antigravity)
*   **Auto-Culling Agent**: An agent that uses Vision API to select the best 30 photos from a raw dump.
*   **Auto-Edit Agent**: An agent that applies Lightroom presets via CLI.
*   **Social Scheduler**: An agent that posts approved content to Client IGs automatically.
